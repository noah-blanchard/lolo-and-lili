# Pet sounds (placeholder slots)

Drop small MP3s here to add audio feedback to pet interactions. Missing files
are safe no-ops (nothing plays). Playback is unlocked by the tap that triggers
it, and respects the in-app mute toggle on the Pet tab.

Expected files:

```
purr.mp3   # petting / grooming / gifting / cuddling
meow.mp3   # feeding / playing / healing
```

Keep them short (<1 s) and quiet; volume is already lowered to 0.5.
